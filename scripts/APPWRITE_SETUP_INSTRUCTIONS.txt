Appwrite manual setup steps (also generated by this script):

1) Start Appwrite (docker compose -f docker-compose.appwrite.yml up -d)
2) Open http://localhost/console
3) Create a project named: lumi-dev
4) Add a Flutter platform with bundle ID: com.lumi.app
5) Enable Email/Password authentication
6) Configure SMTP (use Mailhog in docker-compose for dev):
   Host: mailhog
   Port: 1025
   Secure: false
7) Copy the generated API key into .vscode/mcp.json.template (replace <REPLACE_WITH_API_KEY>)
8) Rename .env.test.template -> .env.test and use with --dart-define-from-file when running integration tests

If Appwrite is reachable, consider using the Appwrite REST API or MCP automation to perform these steps programmatically.
